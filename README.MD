# Lancement :
Depuis nodejs prompt ou un invite de commande avc les paths de node :  
> cd nodejs\myapp  
## Lancer en debug:  
> set DEBUG=myapp:* & npm start  
## Lancer en normal:  
> node ./bin/www  
## Ouvrir avec le naviagteur:  
> http://localhost:8080/  
ou > http://localhost:8080/PIR_website  
ou > votreIP:8080
